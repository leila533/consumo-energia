# ⚡ Calculadora de Consumo de Energia

Projeto simples desenvolvido em **Python** para calcular o consumo mensal de energia elétrica de um aparelho.

---

## 🎯 Objetivo

Ajudar usuários a estimar quanto um aparelho elétrico consome de energia por mês.

---

## ⚙️ Fórmula utilizada

consumoMensal = (potencia × horasDia × 30) ÷ 1000

## ▶ Como executar

1. Instale o Python no computador
2. Abra o terminal na pasta do projeto
3. Execute:

```
python app.py
```

---

## 📊 Exemplo de resultado

Aparelho: Geladeira
Consumo estimado: 45 kWh/mês
Custo estimado: R$ 33,75
